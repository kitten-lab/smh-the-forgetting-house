# The Core Loop
Fugue → Explore → Interpret → Construct Meaning → Alter Future Fugues


### 1. Begin Fugue

- Baseline Emotional State
- Focus Points assigned
- Emotion cards dealt

### 2. Explore the House

- Move between rooms (costs Focus)
- Encounter Matters

### 3. Attempt a Matter

- Combine Emotion + Skill
- Pay Focus
- Success → fragments / skills / focus bonus
- Failure → temporary fragments

### 4. Collect Fragments

### 5. Enter Memory Deck

- Combine fragments → memories
- Combine memories → stories

### 6. Stories Alter Future Runs

- New rooms
- New fragments
- Permanent emotions
- Skills

### 7. Run Out of Focus → Forget

- New Fugue

### 8. House reconfigures

Hand resets

Repeat until Story Pillars filled
